Personal portfolio for Amira Basuony Ammar, a Junior UI/UX Designer focused on
creating simple, intuitive digital experiences in Figma.

## Overview

Single-page portfolio built as a self-contained `index.html` (HTML + CSS + JS,
no build step). Designed with a modern dark theme and a designer-friendly
pink / violet / rose accent palette.

## Sections

- **Hero** — introduction, status, typed tagline, social links
- **Stats** — projects, skills, certifications, languages
- **About** — bio, contact info, focus areas
- **Skills & Tools** — UX research, UX/UI design, prototyping, design systems
- **Services** — what Amira offers, end-to-end
- **Process** — 5-step design process (Discover → Define → Ideate → Design → Prototype & Test)
- **Projects** — Home Repair App, Sleep App
- **Certifications** — Google UX Design Specialization, Figma UI/UX Design Essentials
- **Contact** — email, phone, LinkedIn, Behance, contact form

## Run locally

It's a static page — just open it:

```bash
open index.html
```

Or serve it with any static server:

```bash
python3 -m http.server 8080
# then visit http://localhost:8080
```
